This Bazaarvoice SEO Integration SDK provides you with the tools necessary to deploy a search-friendly implementation of Bazaarvoice hosted displays in Java environments.

Whether you are working on a first-time Bazaarvoice integration, or if you are upgrading the from an older integration, we strongly recommend reading the latest Bazaarvoice SEO Implementation Guide.

Once your integration is complete, don't forget to request the Bazaarvoice SEO Audit by submitting a request at spark.bazaarvoice.com. We will review your implementation and check for typical issues that may prevent search engines from properly indexing your Bazaarvoice-managed content.

Source

The binaries contained in this page should meet the needs for most clients. However, you can find source code on Github at https://github.com/bazaarvoice/seo_sdk_java.

Example

Use the example jsp as a guide. For jsp integrations, the example code can usually be copied and pasted with only a few accout-specific edits. Take note of the embedded comments. View the example jsp.
